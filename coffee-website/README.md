# Breva Coffee — Website Project

**A responsive coffee shop website built with HTML5, CSS3, and JavaScript.**

## Project Structure

```
coffee-website/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles (CSS variables, responsive design)
├── js/
│   └── app.js          # Interactive functionality
└── README.md
```

## Features

- **Responsive Design** — Works on desktop, tablet, and mobile
- **Sticky Navigation** — Scrolls with active section highlighting
- **Hero Section** — Animated intro with call-to-action buttons
- **Marquee Banner** — Scrolling drink names ticker
- **Menu Section** — 12 items with category filter (All / Hot / Cold / Specials)
- **About Section** — Animated coffee cup illustration with stats
- **Contact Section** — Working form with success feedback
- **Toast Notifications** — "Add to order" feedback on menu cards
- **Scroll Animations** — Intersection Observer reveal effects

## Technologies Used

- HTML5 (semantic elements, accessibility)
- CSS3 (Flexbox, Grid, CSS variables, animations)
- Vanilla JavaScript (DOM manipulation, Intersection Observer)
- Google Fonts (Playfair Display + DM Sans)

## How to Run

Simply open `index.html` in any modern browser. No build tools required.
